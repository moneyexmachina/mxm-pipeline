# mxm-pipeline

**Prefect-backed orchestration • CLI/TUI-first • Provenance everywhere • Optional asset sidecars**

`mxm-pipeline` is the orchestration layer for **Money Ex Machina**.
Define flows and tasks in pure Python, run them locally or via Prefect agents, and standardize run provenance (`_meta.json`) plus optional asset metadata (`_asset.json`) for materialized datasets. Ship value now; grow into an asset catalog later—no rewrites.
